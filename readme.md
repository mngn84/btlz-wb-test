## Описание
Приложение ежечасно получает данные по API "Тарифы коробов" Wildberries и сохраняет их в БД. Данные о тарифах накапливаются в БД для каждого дня, обновлясь ежечасно в течение одного дня.
Данные, отсортированные по возрастанию, отправляются в Google Таблицы.

## Запуск
Перед запуском необходимо создать проект в Google Console, зарегистрировать проекте сервисный аккаунт, открыть ему доступ в подключаемых таблицах. Также нужно создать ключ в сервисном аккаунте и скачать json с ним.
Скопируйте в корне проекта .env.example в .env и заполните последний по примеру.

Запуск приложения:
```bash
docker compose up --build
```
Проверка работы:
- Логи
- Данные пояляются в указанных таблицах

Все настройки можно найти в файлах:
- compose.yaml
- dockerfile
- package.json
- tsconfig.json
- src/config/env/env.ts
- src/config/knex/knexfile.ts
